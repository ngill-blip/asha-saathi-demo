Clickable mock-up of the WhatsApp refresher tool

One lesson — TPT counselling: the 5-finger framework — as an ASHA would receive it on her own phone before a home visit. Tap the reply buttons in the chat. The lesson arrives as text + image, with a voice note and a video she can play or download if she wants. Try a wrong quiz answer to see the remediation loop.

Anchored in real WhatsApp features

  • Media messages — voice notes, an image aid, and a short video (works on any WhatsApp).
  • Interactive reply buttons & list menus — the actual WhatsApp Business API building blocks (max 3 buttons / 10 list rows).
  • Every format, one message — text + image inline, with a voice note and video right there to play or download as data allows.
  • Adaptive journey — a wrong answer routes to a remediation message, then a retry.
  • Recognition as messages — stars/badges arrive as chat messages & a sticker, since WhatsApp has no custom scoreboard.

What's real vs simulated: the audio is generated Hindi TTS and the video is an auto-built slide reel — both stand-ins so the team can hear/see the format. A production build would use a recorded regional voice and properly produced video. The chat engine here is a front-end mock, not a live WhatsApp connection.
